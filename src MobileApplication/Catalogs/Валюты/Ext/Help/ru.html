Справочник "Валюты"


Возможности программы отличаются в зависимости от используемой конфигурации: "1С:УНФ" или "1С:Розница". 
Справочник "Валюты" доступен только в приложении "1С:Управление нашей фирмой" (1С:УНФ).



Прикладное решение позволяет регистрировать движение денежных средств в разных валютах.

Валюты, используемых в программе, хранятся в справочнике "Валюты".

Работа с валютами включает два этапа:

  • Заполнение справочника "Валюты";
    • Подбор из Общероссийского классификатора валют (ОКВ);
    • Ввод вручную;
  • Загрузка курсов валют.

 

   

 

Заполнение справочника "Валюты"

Для использования нескольких валют в программе должен быть установлен флажок "Несколько валют" в разделе "Настройки".

Изначально справочник содержит один элемент – российский рубль.

Новые валюты можно подобрать из Общероссийского классификатора валют – ОКВ (рекомендуется) или добавить вручную.

Для валюты заполняется:

  • Наименование валюты – полное наименование валюты.
  • Цифр. код – уникальный цифровой код, служит для идентификации валюты при загрузке курсов из сети Интернет. Например, 840 для доллара США.
  • Симв. код – символьный код валюты, используется в качестве представления в полях ввода. Например, EUR (Евро).
  • Символьное представление – символ, используемый для максимально краткого представления валюты. Например, $ для доллара США.
  • Курс валюты – определяет используемый способ установки курса валют. Курс может вводиться вручную, загружаться из Интернета, рассчитываться на основании другой валюты или по формуле.

Значение курса валют на текущую дату отображается в колонке "Курс" в форме списка справочника валюты.

 

Добавление валюты из Общероссийского классификатора валют (ОКВ)

Прикладное решение предоставляет возможность быстрого заполнения справочника валют путем подбора из Общероссийского классификатора валют (ОКВ).

  1. Нажмите "Добавить".

  2. В форме подбора валют выделите валюту, которую необходимо добавить. Используйте поле "Найти валюту" для быстрого поиска валюты в списке.

  3. Нажмите на кнопку "Выбрать". Новая валюта будет добавлена в справочник.

Следующие поля новой валюты будут заполнены из классификатора автоматически: "Наименование", "Символ", "Символьный код", "Цифровой код".

 Положение переключателя "Курс валюты" будет установлено в положение "загружается из интернета", если возможно загружать курс этой валюты с портала 1С:ИТС.

 

Ввод валюты не из классификатора (вручную)

В некоторых случаях появляется необходимость в использовании в учете внутренней валюты, курс которой может устанавливаться самостоятельно либо рассчитываться автоматически в зависимости от курсов других валют.

Создать такую валюту можно по ссылке "Создать карточку с произвольной валютой", расположенной в форме списка валют.

Сведения о валюте в данном случае заполняются вручную.

 

Загрузка курсов валют

Для пересчета иностранной валюты в рубли используются данные о курсах валют, установленных Центральным банком России.

Курсы валют могут обновляться вручную и автоматически с портала 1С:ИТС, если компьютер подключен к сети Интернет. В приложении в облачном сервисе (например, 1cFresh.com) курсы валют всегда загружаются автоматически.  

Для обновления курсов вручную:

  1. Нажмите кнопку "Загрузить курсы" в списке валют для перехода к загрузке курсов валют. 

  2. Будет предложен список валют, курс которых загружается из Интернета.

  3. Если в списке присутствуют валюты, курс которых обновлять не нужно, снимите флажки напротив них вручную. В форме списка валют следует уточнить период, за который нужно ввести новые курсы, и нажать кнопку "Загрузить и закрыть".

Также в программе предусмотрено регламентное задание "Загрузка курсов валют" для автоматической загрузки курсов валют с портала 1С:ИТС. Для валют, курсы которых обновляются автоматически, должна быть включена настройка "Загружается из Интернета".
